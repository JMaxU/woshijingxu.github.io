保存配资利息方案

url:/future/saveinterest

调用方式

post提交, 编码使用UTF-8;响应json格式输出.

请求参数

字段 数据类型 非空 说明
signMsg string true 验签串
data json true 一个json串

json串格式如下:

contractPrompt 当配资大于所返回的那个数时, 需要给用户提示:
"操盘须知：投资沪深A股，仓位有限制，盈利全归您。 单股不超总操盘资金的50%。"
“5-5”表示第一个代表杠杆, 第二个代表按天还是按月, 如果为0则按天 , 如果不为0, 则是按月, 数字代表是几个月
“1000001.7” 代表在100000之前按每月或每天1.7来算. 配次金额1.7/100 即为每月管理费 或 每天管理费.

响应参数

字段 数据类型 非空 说明

申请配资审核通知接口

url:/future/auditinform

调用方式

post提交, 编码使用UTF-8;响应json格式输出.

请求参数

字段 数据类型 非空 说明
signMsg string true 验签串
orderId string true 网信订单号
rnOrderStatus int true 融牛订单状态
homesAccount string false homes账户
homesPwd string false homes账户密码
auditResult int true 审核结果,0审核通过, 1审核未通过
tradeStartDate sring true 交易开始日期, 格式如:2011-11-11
tradeEndDate sring true 交易结束日期, 格式如:2011-11-11
remarks sring false 备注, 比如审核不通过时的原因

响应参数

字段 数据类型 非空 说明
payId sring false 申请通过后, 转帐时的payId

ftp文件格式 收管理费

融牛

文件每行多个字段, 每个字段有逗号分隔, 每行有如下字段

文件名格式:interest_20150505.csv
md5文件在文件后加后缀.md5 比如interest_20150505.csv.md5
其中md5文件中只存入md5串

字段 数据类型 非空 说明
rnOpOrderId string true 融牛操作订单
orderId int true 网信合约订单号
managementFee int true 管理费金额
date string true 代表哪天的管理费, 格式为:2014-07-08

网信冻结文件

文件名格式:freeze_20150505.csv
md5文件在文件后加后缀.md5 比如freeze_20150505.csv.md5
其中md5文件中只存入md5串

字段 数据类型 非空 说明
rnOpOrderId string true 融牛操作订单
rnUserId string true 融牛用户id
managementFee int true 管理费

成功的不给, 只给失败.

网信扣除管理费文件

文件名格式:deduct_20150505.csv
md5文件在文件名后加后缀.md5 比如deduct_20150505.csv.md5
其中md5文件中只存入md5串

字段 数据类型 非空 说明
rnOpOrderId string true 融牛操作订单
payId string false 有则成功, 没有失败

成功失败都给
